proof of concept
Goal: to demonstrate an auction using metamask to sign transactions for Atomica auction sellers and buyers.
1. requires a local running testnet with the atomica contracts deployed
2. needs a static webpage which shows the auction state
3. assumes a user has a metamask wallet, uses metamask for a faucet transaction to get fakeETH and fakeUSD.
4. user can be a seller, and start an auction.
5. user can be a buyer, and bid on an auction.
6. website shows the countdown to each step of the auction timelines: deadline for seller to deposit fakeETH, deadline for seller to set reserve price, deadline for buyer to place bid, deadline for reveal.
7. auction concludes, and each party is paid out.
